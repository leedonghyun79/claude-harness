## 문서 관리 규칙
- 새 문서 생성 시 이 인덱스에 반드시 추가
- 문서 삭제 시 이 인덱스에서 항목 제거
- 각 문서는 한 줄 설명 필수
- 중복 설명 금지 — 복사하지 말고 링크로 연결
